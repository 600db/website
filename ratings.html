RATINGS

 

Diamond - works perfectly without any tweaks required

Gold - requires minor tweaks to work

Silver - requires extensive tweaking to work

bronze - poor performance but it's technically playbale

wood - completely unplayable / broken

OC / NO OC - overclocked ram or non-overclocked ram

3050 / 3020 - if the game was tested on the 3050e variant or the 3020e variant of the console

auto allocation / XX allocated - how much ram is dedicated to video memory
